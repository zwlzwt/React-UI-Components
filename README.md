# 这是一个基于react的UI组建库，风格采用material design

### 代码风格

Use Airbnb's [javascript code style guide](https://github.com/airbnb/javascript)

### 启动方式

```
$ make run
```

### 代码检查

```
$ make lint
```

### 代码编译

```
$ make build
```

### 代码转移到mock-server-koa2(需要编辑自己的文件路径)

```
$ make real
```

### 涉及技术栈

  * react  
  * webpack >=2.0.0
  * react-router >=4.0.0
